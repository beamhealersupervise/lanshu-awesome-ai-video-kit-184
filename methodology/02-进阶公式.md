# 02 · 进阶公式：8 要素工程化指令

> Seedance 2.0 本质上是一个**多模态 AI 导演**：同时读取文字、图片、视频、音频，并在内部拆成"空间层"（画面里有什么）和"时间层"（事情如何随时间变化）两个维度来理解和生成画面。

好的提示词不是单纯的"文案型形容"，而是"**工程型指令**"：谁、在什么场景、做什么动作、镜头如何运动、按怎样的时间顺序发生。

## 公式

```
精准主体 + 动作细节 + 场景环境 + 光影色调 + 镜头运镜 + 视觉风格 + 画质 + 约束条件
```

简单说就是：先锁定"谁"在"干什么"，再交代"在哪"、"什么氛围"，然后告诉模型"怎么拍"，最后用风格、画质和约束条件把结果收紧。

## 8 要素详解

### 1. 精准主体（Subject）

实际参考的素材里往往存在多个主体，必须明确主体定义。主体可以是人物、道具、场景等。

**基本定义方式**

- 推荐句式：`将 <图片/视频N> 中的 [主体核心特征] 定义为 <主体N>`
- 核心特征要求：使用 **2–3 个清晰、稳定的静态特征**（如服饰、发型、外观、类别）描述，确保唯一可识别性
- 示例：`将图片1中穿红色连衣裙、戴草帽的女人定义为主体1`

**多素材同主体**（多张图指代同一人物）

- 推荐句式：`将图片1中的 […]、图片2中的 […] 定义为 <主体N>`

**多主体场景**（视频中存在多个主体）

- 需分别定义并使用标签区分，标签应具备唯一性与稳定性
- 示例：`将视频1中的高个子男人定义为警察，将另一个矮个子男人定义为小偷`

> ⚠️ 每次涉及主体时需明确指代，避免省略。后续描述中持续使用对应标签（"警察"/"小偷"），避免指代混乱。

### 2. 动作细节（Action）

- **肢体细化 + 程度量化**：动作需具体到手、腿、头部、肩背等部位，同时补充**幅度、速度、力度**描述
  - 示例：缓慢抬手、快速转头、用力蹬地、微微低头
- **优先低缓连续小动作**：尽量规避狂奔、大跳、剧烈翻滚等高爆发动作
- **补充动作过渡衔接**：写明前后动作的惯性与承接关系，保证画面动作连贯自然
  - 示例：借着转身惯性顺势抬手、从停顿状态自然过渡到举手
- **情绪具象外化**：用具体的身体细节表现情绪，替代"很悲伤""非常愤怒"这类抽象词汇（详见 [04-情绪外化表](04-情绪外化表.md)）

### 3. 场景环境（Scene）

明确说明主体所处的场景、位置或空间关系。

### 4. 光影色调（Light）

- **时间**：黄金时刻 / 蓝调时刻 / 正午 / 阴天
- **光源**：硬光 / 柔光 / 窗光 / 霓虹 / 体积光 / 背光
- **色调**：暖调 / 冷调 / 高饱和 / 低饱和 / 电影感分级

### 5. 镜头运镜（Camera）

- **大小**：极广角 EWS / 广角 WS / 中景 MS / 中特写 MCU / 特写 CU / 极特写 ECU
- **运动**：推 / 拉 / 横摇 / 纵摇 / 跟踪 / 环绕 / 手持 / 稳定器 / 航拍
- **角度**：低角度 / 高角度 / 平视 / 鸟瞰

> ⚠️ 一个镜头里**尽量只指定 1 种运镜方式**，不要同时要求推拉摇移，会增加画面的不稳定性。

详见 [05-运镜词典](05-运镜词典.md)。

### 6. 视觉风格（Style）

- **电影感**：cinematic / film grain / anamorphic / 35mm
- **纪录片**：documentary / handheld / naturalistic
- **赛博朋克**：cyberpunk / neon-lit / holographic
- **动漫**：Ghibli-inspired / cel-shaded / anime aesthetic
- **复古**：vintage / 8mm / VHS / retro grain

### 7. 画质（Quality）

定义画面清晰度、细节纹理与光影质感。

- 示例：高清、细节丰富、电影质感、色彩自然、光影柔和

### 8. 约束条件（Negative）

约束词十分重要，可有效规避画面瑕疵、畸形崩坏及不合理元素，约束生成边界与稳定性。

- 避免生成字幕：`保持无字幕` / `避免生成任何文字或字幕`
- 避免生成 Logo：`不要生成Logo`
- 避免生成水印：`不要生成水印`

详见 [06-约束词清单](06-约束词清单.md)。

## 实战示例（来自官方 PDF）

> **场景**：宿舍情感短剧

**素材准备**：
- @图片 1：女主半身照
- @图片 2：宿舍场景参考图
- @视频 1：室内对话运镜参考（中景推拉或轻微摇移）
- @音频 1：室内环境声或轻音乐

**提示词**：
```
@图片 1 中的女孩作为主角，@图片 2 作为宿舍场景风格参考，参考 @视频 1 的运镜方式。

镜头 1：傍晚时分，女孩@图片 1 脚步轻快地走到宿舍门口@图片 2，镜头中景平稳跟拍，
        暖黄色日光从窗外洒进走廊，她在门口停顿一下，深呼吸，表情略带紧张。

镜头 2：女孩@图片 1 推开门走进宿舍，镜头切到室内中景，舍友们一边整理书本一边
        抬头看向她，其中一人笑着问 {考得怎么样啊，过了吗}，镜头在几人之间缓慢
        切换半身特写。

镜头 3：女孩@图片 1 先低头露出落寞表情，镜头给到她的近景，随后她抬头憋不住笑容，
        哈哈大笑说 {骗你们的}，舍友们追着打闹起来，镜头缓慢拉远，定格在宿舍内
        一片欢声笑语的全景画面。

全程画面高清电影纪实风，色调温暖，光影柔和；人物面部稳定不变形，动作自然流畅，
无卡顿无闪烁；环境音效与 @音频 1 自然融合。
```
