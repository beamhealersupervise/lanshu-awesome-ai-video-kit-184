# 01 · 基础公式：选对任务类型

> 来源：火山方舟官方《Doubao Seedance 2.0 系列提示词指南》

Seedance 2.0 支持多模态素材（图片 / 视频 / 音频），写提示词前**先想清楚你做的是哪种任务**，每种任务的句式都不一样。

## 四种任务类型

### 1. 多模态参考（生成全新视频）

从素材中提取部分元素（主体 / 风格 / 场景 / 音效），生成一个全新视频。

- **适用场景**：动作迁移、主体复用、氛围借鉴
- **推荐句式**：
  - 图片参考：`参考 <图片N> 中的 <主体N>，生成…`
  - 视频参考：`参考 <视频N> 中的 <动作/运镜/风格/音效>，生成…`
  - 音频参考：`参考 <音频N> 中的音色，生成…`

### 2. 编辑视频（在原视频上局部/全局修改）

未提及的部分默认保持不变。

- **适用场景**：局部替换、主体抹除、属性修改
- **推荐句式**：
  - 增加元素：清晰描述 `<元素特征>` + `<出现时机>` + `<出现位置>`
  - 修改元素：严格编辑 `<视频N>`，将其中的 `<原特征>` 修改为 `<新特征>`
  - 删除元素：点明需要删除的元素，对于保持不变的元素，在提示词中加以强调，表现更佳

### 3. 延长视频（在时间维度上续写）

要求音视频风格、主体与叙事保持一致。

- **适用场景**：续写剧情、延展动作、补全片段
- **推荐句式**：
  - 延长视频：`向前/向后延长 <视频N>，生成…`
  - 轨道补全：`<视频1> + <过渡画面描述> + 接 <视频2> + <过渡画面描述> + 接 <视频3>`

> ⚠️ 编辑 / 延长视频任务直接使用 `<视频N>` 指代视频，**请勿使用 "参考 <视频N>"**，以免被误判为参考任务。

### 4. 组合任务

上述 3 种任务也支持组合使用。

- **适用场景**：参考某个素材，编辑另一个素材
- **推荐句式**：`参考 <图片/视频N> 的 [参考维度]，严格编辑 <视频X>，[具体编辑内容]`

## 怎么选？决策树

```
你手上有什么素材？
├── 只有文字描述                     → 文生视频（直接写进阶公式）
├── 有图/视频/音频，想做新视频       → 任务 1：多模态参考
├── 已有视频，想改/加/删元素         → 任务 2：编辑视频
├── 已有视频，想往前/往后续写         → 任务 3：延长视频
└── 上面 2-3 个都要                  → 任务 4：组合任务
```

## 限制

- Seedance 2.0 系列模型**最多支持 3 段视频输入**，总时长不得超过 **15 秒**
- 生成时将自动截取首尾视频的衔接部分，仅保留必要片段参与合成
